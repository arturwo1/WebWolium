1. Introduction

This Privacy Policy explains how Wolium Bot ("Bot", "Service") collects, uses, stores, and protects user data when users interact with the Bot through Discord or connected services.

By using the Bot, you agree to this Privacy Policy.

2. Data Controller

Data Controller: Arturwolium (Independent Developer)

Location: Lithuania, European Union

Contact: Discord Support Server

3. Data We Collect

3.1 Discord Data

  • User ID
  • Guild / Server ID
  • Channel ID
  • Command usage data
  • Error logs
  • Message metadata (timestamps, message links, server/channel/user IDs)
  • Voice activity metadata (join time, previous/current channel, server ID, user ID)

3.2 Message Content

Full message content is stored only if explicitly enabled or requested by the user.

3.3 Attachments

Only attachment links stored by Discord may be stored. The Bot does not store actual uploaded files.

3.4 OAuth Data

  • Discord OAuth authentication data
  • Server list accessible to the user
  • Access tokens
  • Refresh tokens

4. How We Use Data

  • Provide bot functionality
  • Process commands
  • Improve stability and performance
  • Provide AI-based features
  • Prevent abuse and ensure security

5. AI Processing (Third-Party Processing)

Some features use third-party AI services via HuggingFace API.

Data that may be sent to AI processing includes:

  • Message content (only if enabled by user)
  • Server / channel / username
  • Permissions data
  • Message timestamps
  • Reply context

AI providers may temporarily process data only to generate responses.

6. Data Storage

  • Database Provider: Supabase
  • Region: European Union (eu-central-1)
  • Data transmission: Encrypted via HTTPS
  • Sensitive tokens: Encrypted at rest

7. Data Retention

Data is stored until:

  • User requests deletion, OR
  • Data is no longer required for service operation

8. Data Deletion

Users can request deletion of all stored data using a bot command.

Currently, deletion removes all stored logs, message history, and voice activity records related to the user.

9. Internal Logging

Some logs may be stored in private internal Discord channels for debugging and moderation purposes.

Access is limited to bot administrators and staff only.

10. Third-Party Services

  • Discord (Platform Provider)
  • Supabase (Database Infrastructure)
  • HuggingFace (AI Processing API)

11. Security Measures

  • Encrypted communication (HTTPS)
  • Encrypted storage of sensitive tokens
  • Restricted database access
  • Access limited to authorized bot staff

12. User Rights (GDPR)

Users located in the European Union may have the right to:

  • Request access to stored data
  • Request deletion of data
  • Object to certain processing
  • Request correction of incorrect data

13. Children

The Service is not specifically directed toward children. If a parent or guardian believes data was collected from a minor, they may contact the Data Controller.

14. Changes to This Policy

The Privacy Policy may be updated periodically. Significant changes may be communicated via the Bot or support server.
